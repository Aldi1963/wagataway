PANDUAN PEMASANGAN CPANEL - WA GATEWAY SAAS PRO

### 1. Upload dan Extract File
1.  Login ke cPanel Anda.
2.  Buka File Manager.
3.  Masuk ke direktori Home (biasanya /home/username/).
4.  Buat folder baru bernama "wagateway" (jangan di dalam public_html).
5.  Upload file ZIP ini ke dalam folder "wagateway" tersebut, lalu Extract.

### 2. Buat Database PostgreSQL
1.  Cari menu "PostgreSQL Databases" di cPanel.
2.  Buat database baru, contoh: user_wagateway.
3.  Buat user baru dan catat passwordnya.
4.  Hubungkan User ke Database dengan hak akses "ALL PRIVILEGES".
5.  Catat URL koneksinya: postgresql://user:pass@localhost:5432/user_wagateway

### 3. Konfigurasi Environment (.env)
1.  Di File Manager, buka folder wagateway/artifacts/api-server/.
2.  Edit file ".env". Sesuaikan detail database Anda:
    DATABASE_URL=postgresql://user:pass@localhost:5432/user_wagateway
    SESSION_SECRET=buat_random_string_panjang
3.  Simpan (Save).

### 4. Setup Node.js App di cPanel
1.  Buka menu "Setup Node.js App" di cPanel -> Create Application.
2.  Node.js version: 20.x ke atas.
3.  Application mode: Production.
4.  Application root: wagateway
5.  Application URL: domain-anda.com
6.  Klik Create.

### 5. Install Module & Database (Lewat Terminal)
1.  Buka menu Terminal di cPanel.
2.  Paste perintah "Command for entering virtual environment" yang ada di halaman Setup Node.js tadi.
3.  Jalankan:
    cd ~/wagateway
    npm install -g pnpm
    pnpm install
    pnpm --filter @workspace/db run push

### 6. Jalankan API Server (PM2)
Di Terminal cPanel:
    pnpm install -g pm2
    pm2 start artifacts/api-server/dist/index.mjs --name "wa-api"
    pm2 save

### 7. Konfigurasi .htaccess
Edit file .htaccess di public_html Anda dan tambahkan:

RewriteEngine On
RewriteCond %{REQUEST_URI} ^/api/ [NC]
RewriteRule ^(.*)$ http://127.0.0.1:8080/$1 [P,L]
RewriteCond %{DOCUMENT_ROOT}/wagateway/artifacts/wa-gateway/dist/public%{REQUEST_URI} -f
RewriteRule ^(.*)$ /wagateway/artifacts/wa-gateway/dist/public/$1 [L]
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^ /wagateway/artifacts/wa-gateway/dist/public/index.html [L]
